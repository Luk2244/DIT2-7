ขั้นตอนการสร้าง project

1.ไปที่สัญลักษณ์ เลือก New Repository ตั้งค่าใส่ชื่องานรายละเอียด กด Create Repository
2.คลิก Add file - upload files - choose your files - เลือก commit directly to the main branch คลิก Commit changes ตั้งค่า Create ใส่ชื่อ Repository ใส่รายละเอียด Description
ตั้งค่าความเป็นส่วนตัวเป็น Public
ตั้งค่าให้สามารถเพิ่มไฟล์ได้ที่ Add a READEME file

เพิ่มเพื่อนเข้ากลุ่ม Git
3.เพิ่มเพื่อนให้สามารถเข้าถึง Project โดยไปที่ Setting เลือก Colloborators แล้วกด Add people และค้นหาชื่อเพื่อน(ชื่อผู้ใช้)
4.ถ้าต้องการจะออกจากกลุ่มงานให้ไปที่ไอคอน Profile ผู้ใช้งานเลือก Settings ไป Repository จะปรากฏกลุ่มที่อยู่ทั้งหมดสามารถกด Leave
เพื่อออกจากกลุ่มที่ต้องการ

เชื่อม Git เข้า vs code
5.เชื่อมต่อ Git โดยการเข้าไปใน vs code ไปที่ source Control
เลือก Dowload Git for windows เลือก Git for windows x64 setup
6.เมื่อดาวโหลดเสร็จกด Accounts - Signin to use Copilot ลงทะเบียนใช้ Git
