ขั้นตอนการสร้าง Project

1. เข้าGitHub แล้ว Login ไอดีที่ได้สร้างไว้ แล้วกด Create New แล้วคลิกที่ New repository ให้ใส่รายละเอียดเช่นชื่องาน แล้วกด create repository

2. กดที่ Add file แล้วกด Upload file ให้เลือกว่าจะะลากไฟล์ลงมา หรือกดเข้าไปเลือกไฟล์เอง แล้วกด Commit changes 

ขั้นตอนการเพื่มเพื่อน

3. กด settings แล้วไปที่ Collaburators แล้วกด Add people แล้วให้ใส่ชื่อเพื่อนเพื่อดึงเข้ากลุ่ม แล้ว กด Add to pepository

ขั้นตอนการออกจากกลุ่ม

4. การออกจากกลุ่มให้กดที่โปรไฟล์ตัวเอง แล้วกด Setting แล้วเลื่อนลงไปกดที่ Repositories แล้วกด Leave กลุ่มที่ต้องการออก

เขื่อมต่อGitกับVS code

5. เข้า VS code แล้วโหลด Git Hub โกยเข้าไปทื่ source Control แล้วกด Download Git for Windows เลือก Git for Windows x64 Setup

6. เมื่อดาวโหลดเสร็จให้กด Accounts - Sign in to use Copilot ลงทะเบียน Git
