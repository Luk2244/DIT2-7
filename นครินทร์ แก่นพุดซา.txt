การสร้าง Repository

1.ลากเมาส์ไปที่ปุม + บนขวาของจอ Create new...

2.กดไปที่ New Repository
2.1)หลังจากนั้นให้เลือก Owner
2.2)ให้ตั้งชื่อRepository name
2.3)สามารถเขียนคำอธิบายได้ Description(ตัวเลือก)
2.4)เลือกได้ว่าจะตั้งงานเป็นสาธารณะ หรือ ส่วนตัว Public/Private
2.5)สามารถเพิ่มไฟล์ README ได้

3.เมื่อเสร็จสิ้นแล้วกดคำว่า Create repository

4 เสร็จแล้วกดคำว่า Add file เพื่อเพิ่มไฟล์งานของตัวเอง
4.1)สามารถสร้างไฟล์งานได้ + Create new file
4.2)สามารถอัพโหลดไฟล์ได้ ^ Upload file
